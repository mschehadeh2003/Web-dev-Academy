JavaScript Tutorial

Introduction to JavaScript

JavaScript is a programming language that adds interactivity to your websites. Let's start with the basics!

// Variables and Data Types let name = "John"; const age = 25; var isStudent = true; // Printing to console console.log("Hello, " + name); // Modern string interpolation console.log(`${name} is ${age} years old`);

Working with the DOM

The Document Object Model (DOM) allows JavaScript to interact with HTML elements.

This text will change when you click the button.

// DOM Manipulation Example function changeText() { document.getElementById("demo").innerHTML = "Text changed!"; }

jQuery Basics

jQuery is a JavaScript library that makes it easier to work with the DOM.

This text will fade out and in.

// jQuery Example $(document).ready(function() { $("#jqueryBtn").click(function() { $("#jqueryDemo").fadeOut().fadeIn(); }); });
