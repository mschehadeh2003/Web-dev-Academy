CSS Tutorial

Basic CSS Syntax

CSS (Cascading Style Sheets) is used to style HTML elements. Here's the basic syntax:

/* Selector and Declaration Block */ selector { property: value; another-property: value; } /* Example */ .my-class { color: blue; font-size: 16px; margin: 10px; }

CSS Box Model

Every HTML element is a box with content, padding, border, and margin:

.box { width: 100px; height: 100px; padding: 20px; background-color: blue; border: 2px solid black; margin: 10px; }

Flexbox Layout

Flexbox makes it easy to create flexible layouts:

Item 1
Item 2
Item 3
.container { display: flex; justify-content: space-between; align-items: center; } .item { flex: 1; margin: 10px; }

Media Queries

Media queries allow you to create responsive designs:

/* Mobile First Approach */ .container { width: 100%; padding: 10px; } /* Tablet and larger */ @media (min-width: 768px) { .container { width: 750px; margin: 0 auto; } } /* Desktop */ @media (min-width: 1024px) { .container { width: 960px; } }
